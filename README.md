# Tiempo Pro ⏱️

Aplicación local para cálculo de intervalos de tiempo y cronómetro profesional.

## Características
- Calculadora de suma y resta de horas, minutos y segundos.
- Cronómetro con vueltas.
- Cuenta regresiva.

## Requisitos
- Python 3.x
- Flask

## Instalación
1. Instalar dependencias: `pip install -r requirements.txt`
2. Ejecutar: `python app3.py`
